Movie Finder – React IMDb Explorer
A single‑page web application built with React that allows users to search through a massive movie database powered by an IMDb API.
Users can explore movies, rate them, and maintain a personal list of watched films.

--Features--
Summary of movies you watched (Time , your rate , IMDb rate)
localStorage
Movie details
Search movies from a large IMDb-based movie database
View movie details
Rate movies bt your own from 1 to 10
Add movies to your personal Watched List
Clean single-page experience with React
Fast search interaction with external API
Clean UI

--How It Works--
Use the search bar to find any movie.
The app fetches results from the IMDb API.
Click on a movie to see more details.
Give the movie a rating.
Add it to your Watched Movies list to track what you’ve already seen.

--Installation--
Install dependencies:
npm install

Run the development server:
npm run start

The app will start locally and you can open it in your browser and all movies you added will be save at localhost
